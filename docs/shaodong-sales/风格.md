UI风格的几个核心要点

**结构**：用 `section-title`（小号全大写灰色标签）+ 内容块的组合，信息层级一眼清晰，不靠大标题撑场面。

**卡片**：白底、0.5px 细边框、12px 圆角，有质感但不抢眼。卡片只做容器，不加阴影、不加色块背景。

**颜色克制**：正文只用 `#111` / `#555` / `#888` 三档灰，彩色只出现在 tag 上，且每个 tag 用同色系的浅背景+深字色，不用纯色块。

**字号收敛**：section-title 12px、card-title 15px、card-body 14px，整体偏小，密度高但不压迫。字重只用 400/500，不用 600+。

**间距靠 margin 而非 padding**：卡片之间 10px gap，section 之间 2rem，节奏感来自留白的韵律而不是分割线。

**功能性装饰**：步骤编号圆圈、tag 色块、分割线 `<hr>`，每个装饰元素都在传递信息，没有纯装饰。



把CSS以及对应的标签规范固定下来复用

字色三档：`#111` / `#555` / `#888`

卡片：白底 + `0.5px #e0e0e0` 边框 + `12px` 圆角，无阴影

字号：section-title 12px、card-title 15px、body 14px

间距：section 间 `2rem`，卡片间 `10px`

彩色只用在 tag/badge，且必须是浅底深字

复用规范速查

| CSS 类                                         | 对应节点                  | 用途                                 |
| ---------------------------------------------- | ------------------------- | ------------------------------------ |
| `.outer`                                       | `<div>` 最外层            | 居中限宽 + 移动端全屏                |
| `.wrap`                                        | `.outer` 内第一个 `<div>` | 内边距容器                           |
| `.section`                                     | `<div>`                   | 每个内容块，`margin-bottom: 2rem`    |
| `.section-title`                               | `<div>`                   | 灰色小标题，12px 全大写              |
| `.highlight`                                   | `<div>`                   | 灰底强调块，用于原则/结论            |
| `.card`                                        | `<div>`                   | 白底细边框卡片，通用容器             |
| `.card-title`                                  | `<div>`                   | 卡片标题，15px 500                   |
| `.card-body`                                   | `<div>`                   | 卡片正文，14px #555                  |
| `.row`                                         | `<div>`                   | 两列 grid，移动端自动单列            |
| `.divider`                                     | `<hr>`                    | 0.5px 分割线                         |
| `.badge`                                       | `<span>`                  | 小标签，配色变体用 `.badge-*`        |
