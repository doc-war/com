处理异议的原则
客户说"不需要"，99%不是真的不需要，是没看到和他有关的理由。你的工作是帮他想清楚，不是说服他。
高频异议
客户说
我们不需要官网，客户都是熟人介绍的
背后是
觉得官网和自己的获客方式无关，看不到直接价值
怎么说
"熟人介绍过来之后，对方第一件事是不是搜一下你们？搜不到，或者搜到的东西很杂，这单可能就黄了。官网不是用来找客户的，是让别人确认你靠谱的。"
不要否定他的获客方式，顺着说——官网是熟人介绍链条里的最后一道信任关
客户说
太贵了 / 现在不想花这个钱
背后是
不确定值不值，在规避风险
怎么说
"我们是先做出来你看，满意了再付。你现在不需要花钱，看了不满意就算了，不会有损失。"
把"花钱"的时间点往后推，他的阻力会小很多。不要在价格上纠缠，强调零风险。
客户说
我在XX平台上有店，够用了
背后是
已经有线上存在感，觉得不需要再投入
怎么说
"平台上的店是平台的，客户搜你公司名的时候找不到。官网是你自己的门面，两个不冲突，而且官网能把平台的流量接住。"
不要否定平台价值，本质是"互补"而不是"替代"关系。
客户说
我已经做了一个官网了
可能真相
如果客户不懂，可能是被骗的状态
怎么说
"方便转我官网地址或二维码吗，我可以帮你分享给周边的朋友，或者帮你做个二维码。"
确认实际情况：1、是小程序卡片形式的伪官网还是真的浏览器官网。2、是否很老旧。3、能否百度全名搜索第一页有
客户说
做了也没用，没人会搜到我们
背后是
对官网的效果没信心，觉得是白花钱
怎么说
"主动搜你的人本来就不多，但你把官网链接发给客户、印在名片上、放在报价单里，对方点进去看到的是什么，差别很大。"
把"搜索引流"这个预期拉低，改成"信任背书"——这才是官网对工厂客户的真实价值。
客户说
我再想想 / 过段时间再说
背后是
没有紧迫感，或者还有顾虑没说出来
怎么说
"没问题，你先给我公司名和联系方式，我回去帮你查一下网上现在能不能搜到你们，结果发给你，你再决定也不迟。"
不要就这样走了。低门槛动作拿到信息，制造下次联系的理由，同时让他感受到"已经在帮他做事"。
客户说
你们做出来的会不会很模板、很廉价
背后是
见过烂官网，对品质有顾虑
怎么说
"你先看看我们做过的，不好看你直接说，反正不满意不收钱。"（翻出案例给他看，截图+网址或二维码）
不要口头解释"我们很专业"，直接给他看。案例是最好的回答。
客户说
你们公司介绍的PPT有没有，案例相关的都给我们评估下
背后是
被传统的套路流程忽悠瘸了，他担心被坑
怎么说
"我们是本地服务，不搞这一套，过去PPT忽悠签合同、先预付费再开发再付尾款这套，不是我们的服务方式。"
不要顺着客户这个要求走，资质不是小公司的优势维度，关注点在服务形式，低价、1天见草稿、不用预付已经是体验天花板。
真正的死单长什么样
不是客户说"不需要"，而是：老板完全不上网、业务本身说不清楚、或者对方明确说过"网上的东西我不信"。遇到这三种，礼貌收场，不要死磕。
